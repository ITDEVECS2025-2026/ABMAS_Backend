# ABMAS_Backend
backend ini akan hidup di server, backend ini akan include database dengan PostgreSQL, object relation mapping menggunakan Prisma, server menerima input data dari hardware SIM7600, komunikasi hardaware dan server dengan menggunakan protokol HTTP(Socket.io, http), Backend ini menggunakan payload : JSON{}->route->controller->service->HTTP Socket(subs, unsubs), backend ini nanti ke PC, didalam PC terdapat Domain dan Host. Server ini hidup di dalam PC, dari Server ini terhubung ke frontend aplikasi berisi interface data. Server dan aplikasi saling subscribe, IP Public (Domain)
